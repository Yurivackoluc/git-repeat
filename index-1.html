Инициализировать репозиторий git init

возврат cd ..

git add +файл или git add . добавить файл в репозиторий или все

git commit -m"назв ветки"

Чем чаще делаем коммит тем больше мы можем вернуться назад.

.gitignore - игнорить файлы и папки

git branch - в какой ветке находимяс (мастер или др)

git branch test создание ветки test-

git checkout test вход в ветку тест

git checkout -d test удаление ветки тест

git chechout -b new создание новой ветки new и сразу переход на нее

.gitignore (для хранения файлов не нужн проек)

/ + назв файла чтоб положить в .ginignorе

git merge - обединение с веткой

git config --global config user.name (имя кот используем)

git config --global config user.email (почта кот используем)

поменять имя/почту дописываем после user.name имя что будем использовать
